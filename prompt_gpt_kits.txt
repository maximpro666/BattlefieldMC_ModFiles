Создай JSON-конфиг для системы китов в Minecraft-моде BattlefieldMC (TeamSystem).

## Формат файла
Файл будет называться `kits.json` и лежать в `world/teamsystem/kits.json` на сервере.

## Структура JSON
```json
{
  "classes": {
    "уникальный_id_класса": {
      "display_name": "Название класса (локализованное)",
      "icon": "эмодзи-иконка",
      "kits": {
        "уникальный_id_кита": {
          "display_name": "Название кита",
          "description": "Описание (1 фраза)",
          "weapons": {
            "primary": ["id1", "id2", ...],
            "secondary": ["id1", "id2", ...],
            "special": ["id1", "id2", ...],
            "grenade": ["id1", "id2", ...]
          },
          "armor": {
            "helmet": ["warborn:nato_helmet", "warborn:nato_helmet_woodland"],
            "chestplate": ["warborn:nato_shturmovik_chestplate"],
            "backpack": ["warborn:nato_sqad_leader_backpack"],
            "shoulderpads": ["warborn:nato_shoulderpads"]
          },
          "requirements": {
            "rank": число_от_1_до_10,
            "team": "NATO" или "RUSSIA"
          }
        }
      }
    }
  }
}
```

## Правила
1. **Каждый класс должен быть привязан к команде** через `requirements.team` (NATO или RUSSIA)
2. **Ранги** (rank): 1 — новобранец, 10 — генерал. Базовые киты — rank 1, элитные — rank 3-5
3. **Оружие** — список ID, игрок выбирает из них. Первый ID = вариант по умолчанию

## Локализация (двуязычность)
На сервере есть настройка языка (`ru` или `en`). Все названия и описания должны поддерживать оба языка.

Формат: **display_name** и **description** пиши через разделитель ` // `: сначала английский, потом русский.

Если язык сервера английский — покажется текст до ` // `. Если русский — текст после ` // `.

Примеры:
- `"display_name": "Assault // Штурмовик"`
- `"description": "Frontline assault rifleman // Пехотинец штурмового отряда"`

Для NATO-классов английская часть идёт первой. Для Russia — тоже английская часть первой, русская после ` // ` (чтобы английский был основным, русский — локализацией).

## Требования к результату
1. **Минимум 6 классов** (по 3 на каждую команду)
2. **У каждого класса 2-3 кита** (базовый, продвинутый, элитный) с разными рангами
3. **В каждом ките минимум 2 варианта primary** (чтобы был выбор)
4. **Комбинировать TACZ + Superb Warfare** в одном ките (например primary: `["tacz:m4a1", "superbwarfare:m_4"]`)
5. **Использовать аддоны** (daffas_arsenal, maxstuff) где уместно
6. **Все display_name и description — двуязычные** через ` // ` как описано выше
7. **Баланс**: одинаковые по смыслу киты для NATO и Russia должны быть на одном ранге

## Пример одной записи
```json
"assault_nato": {
  "display_name": "Assault // Штурмовик",
  "icon": "⚔",
  "kits": {
    "rifleman": {
      "display_name": "Rifleman // Стрелок",
      "description": "Standard frontline rifleman // Стандартный пехотинец",
      "weapons": {
        "primary": ["tacz:m4a1", "tacz:m16a4", "superbwarfare:m_4", "daffas_arsenal:hk416"],
        "secondary": ["tacz:glock_17", "superbwarfare:m_1911"],
        "special": [],
        "grenade": ["tacz:m67", "superbwarfare:hand_grenade"]
      },
      "requirements": { "rank": 1, "team": "NATO" }
    }
  }
}
```

Верни ТОЛЬКО JSON, без пояснений. Файл должен быть валидным JSON и готов к использованию.
